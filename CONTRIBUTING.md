# Contribution Guidelines

First off, thank you for considering contributing to Awesome Meteorology! It's people like you that make this a great tool for the community.

## How to Contribute

To keep this list highly curated and easy to read, please follow these simple guidelines when submitting a Pull Request (PR):

* **Search first:** Check the `README.md` to make sure the resource hasn't already been added.
* **Keep it relevant:** Ensure the resource is directly related to meteorology, atmospheric science, or weather forecasting.
* **Check your links:** Make sure the URL resolves correctly and doesn't lead to a 404 or a paywall.
* **Format correctly:** Use the standard format for your addition:
  `* [Resource Name](URL) - A short, one-sentence description of what it does.`
* **Alphabetize:** Place your new entry in alphabetical order within its specific category.
* **One PR per resource:** If you have multiple resources to add, please submit them as separate Pull Requests (unless they are part of a specific theme).

## Pull Request Process
1. Fork the repository.
2. Add your link following the format above.
3. Submit a Pull Request with a brief explanation of why this resource is useful. 

Thank you for helping build the ultimate meteorology directory!
